# 知识笔记模板

独立概念笔记（非课程 daily），用于知识卡片速查。

## 结构规范

```markdown
# 概念名

## 🎯 核心概念

### 1. 基础定义
- **类型**：分类/回归/优化/...
- **适用场景**：什么问题用这个
- **一句话定义**：...

### 2. 数学原理
- 核心公式
- 公式直觉（人话解释）

### 3. 实践要点
- 参数怎么选
- 常见坑
- 与其他概念的关系

## 📊 关键公式

| 公式 | 含义 | 直觉 |
|------|------|------|
| ... | ... | ... |

## 🔗 关联概念
- 前置知识：...
- 后续应用：...
- 同类对比：...

## 💡 记忆技巧
用一句话或类比帮助记住这个概念
```

## 质量底线

- 文件大小 > 500 bytes
- 必须包含公式表格
- 必须说明与其他概念的关系
