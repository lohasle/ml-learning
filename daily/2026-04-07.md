# 第1课：线性回归与最小二乘法

## 今日学习主题
**线性回归：从数据中找到最佳拟合直线**

## 为什么从线性回归开始？
线性回归是机器学习最基础的算法，也是最直观的。它回答一个简单问题：给定一组数据点，哪条直线能最好地描述它们的关系？理解了线性回归，就理解了"模型学习"的本质——用数据调整参数。

## 这个知识点在 AI 演进中的位置
- **数学基础**：最小二乘法由 Legendre（1805）和 Gauss（1809）独立提出
- **统计学习基石**：所有监督学习算法的基础范式
- **现代应用**：预测分析、趋势预测、特征关系建模

## 核心概念解释

### 1. 什么是线性回归？
用一条直线（或超平面）来拟合数据：
```
y = kx + b
```
- `k`：斜率（权重/系数）
- `b`：截距（偏置）
- 目标：找到最优的 `k` 和 `b`

### 2. 损失函数：最小二乘法
如何衡量"拟合得好不好"？用**均方误差（MSE）**：
```
L = (1/n) * Σ(yi - (k*xi + b))²
```
- `yi`：真实值
- `k*xi + b`：预测值
- 目标：让 L 尽可能小

### 3. 梯度下降求解
不能直接算出最优解时，用**梯度下降**迭代逼近：
```
k_new = k_old - α * ∂L/∂k
b_new = b_old - α * ∂L/∂b
```
- `α`：学习率（步长）
- 每一步都让损失变小
- 直到收敛（梯度接近 0）

### 4. 正规方程（解析解）
对于线性回归，可以直接用公式求出最优解：
```
θ = (X^T X)^(-1) X^T y
```
- 优点：一步到位，不需要迭代
- 缺点：矩阵求逆计算量大（特征多时慢）

## 关键公式直觉

| 概念 | 公式 | 直觉 |
|------|------|------|
| 预测函数 | ŷ = kx + b | 用直线拟合数据 |
| 损失函数 | L = Σ(y-ŷ)² | 预测离真实值越远，惩罚越大 |
| 梯度下降 | θ -= α·∂L/∂θ | 沿着下坡方向走 |
| 学习率 α | 步长 | 太大跳过最优，太小收敛慢 |

## 动手实践要点

### 用 NumPy 从零实现
1. 初始化参数 k=0, b=0
2. 循环 N 次：
   - 计算预测值 ŷ = k*x + b
   - 计算损失 MSE
   - 计算梯度 ∂L/∂k, ∂L/∂b
   - 更新参数
3. 观察损失曲线是否收敛


## 常见问题

**Q: 为什么用平方误差而不是绝对值误差？**
A: 平方误差处处可导，方便梯度计算；同时对大误差惩罚更重，促使模型减少离群值的影响。

**Q: 学习率怎么选？**
A: 经验值 0.001-0.1。可以先用较大值，观察损失是否震荡，震荡就调小。

**Q: 特征需要标准化吗？**
A: 建议标准化。不同量纲的特征会让梯度下降的路径呈椭圆形，收敛慢。标准化后路径更圆，收敛快。

## 本课学到什么

1. **模型 = 参数化函数**：线性回归用 y=kx+b 建模
2. **学习 = 优化参数**：通过最小化损失函数找到最优参数
3. **梯度下降 = 通用优化器**：不仅用于线性回归，几乎用于所有深度学习
4. **从零实现很重要**：sklearn 一行代码搞定，但自己实现才能理解本质
