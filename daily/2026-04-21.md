# 2026年4月21日 - 第13课：CNN 与图像识别（MNIST）

## 今日学习主题
**从全连接到卷积——理解 CNN 如何用「局部感知 + 参数共享」彻底改变图像识别。**

## 为什么学这个？

上一课我们学了优化器和正则化，掌握了训练神经网络的工程技巧。但到目前为此，我们用的都是**全连接网络**——每个神经元和上一层所有神经元相连。

全连接网络处理图像有两个致命问题：
1. 参数爆炸（一张 28×28 的图就有 784 个输入）
2. 丢失空间结构（图像拉成一维向量后，相邻像素的关系消失了）

CNN（Convolutional Neural Network）就是为解决这两个问题而生的。它是计算机视觉领域的基石，也是深度学习「第一次大爆发」的功臣——2012 年 AlexNet 用 CNN 赢得 ImageNet 比赛，误差率从 26% 暴降到 15%，直接引爆了深度学习革命。

**在 AI 演进中的位置**：CNN 是深度学习从理论走向实用的转折点。没有 CNN，就没有后来的图像识别、自动驾驶视觉、医学影像分析。理解 CNN 是理解现代视觉 AI 的前提。

## 核心概念解释

### 1. 卷积操作（Convolution）

**直觉**：想象你用一个放大镜（3×3 的小窗）在照片上从左到右、从上到下滑动，每到一个位置就做一次「加权求和」。这个小窗就是**滤波器（filter/kernel）**。

- 不同滤波器检测不同模式：有的检测横向边缘，有的检测纵向边缘
- 一个 3×3 的滤波器只有 9 个参数，却能扫描整张 28×28 的图
- 这就是**参数共享**——同一套参数在不同位置复用

**关键参数**：
| 参数 | 含义 | 典型值 |
|------|------|--------|
| kernel_size | 滤波器大小 | 3×3, 5×5 |
| stride | 滑动步长 | 1, 2 |
| padding | 边缘填充 | 0 (valid), 1 (same) |
| out_channels | 滤波器数量 | 16, 32, 64 |

### 2. 池化操作（Pooling）

**直觉**：把 2×2 的区域压缩成 1 个值（取最大）。相当于给图像「降采样」。

作用：
- 减少数据量（28×28 → 14×14）
- 增强平移不变性（猫稍微移动几个像素，池化后差异不大）
- 没有可学习参数，纯数学操作

### 3. 典型 CNN 架构

```
输入图像 → Conv → ReLU → Pool → Conv → ReLU → Pool → Flatten → FC → Softmax → 预测
(28×28)  (特征提取)                  (降维)          (分类决策)
```

**设计哲学**：前面的卷积层负责「看」（提取特征），后面的全连接层负责「想」（做出分类决策）。

## 关键公式直觉

| 概念 | 公式 | 直觉解释 |
|------|------|----------|
| 卷积输出尺寸 | `(H - K + 2P) / S + 1` | 输入大小减去滤波器大小，加上填充，除以步长 |
| 参数量（卷积层） | `C_out × C_in × K² + C_out` | 每个输出通道对应一个滤波器 × 输入通道数 |
| 最大池化 | `max(region)` | 取局部最强响应，保留最显著特征 |
| ReLU | `max(0, x)` | 负值清零，引入非线性 |

## 常见问题

**Q: 卷积和互相关有什么区别？**
A: 严格来说，深度学习中的「卷积」其实是互相关运算（不翻转滤波器）。数学上的卷积会先 180° 翻转滤波器再滑动求和。但因为滤波器参数是学出来的，翻不翻转都一样能学到等价的结果。

**Q: 为什么 CNN 的滤波器能自动学到有用的特征？**
A: 反向传播！损失函数通过梯度告诉每个滤波器参数该怎么调。经过大量样本训练，滤波器自然学会检测边缘、纹理等对分类有帮助的模式。这正是深度学习「自动特征学习」的精髓。

**Q: 1×1 卷积有什么用？**
A: 改变通道数而不改变空间尺寸。在 Inception 和 ResNet 等现代架构中用于降维/升维，相当于逐像素的全连接层。

## 本课总结

1. CNN 的核心 = 卷积（局部感知 + 参数共享）+ 池化（降维 + 不变性）
2. 卷积层自动学习特征（边缘→纹理→部件→整体），越深层越抽象
3. 相比全连接，CNN 参数更少、效果更好、训练更快——是图像任务的标准选择
4. 2012 年 AlexNet 的成功标志着深度学习时代的开始
