## 核心概念解释

### 1. 为什么需要评估
- **避免过拟合**：模型可能记住了训练数据但泛化能力差
- **模型选择**：在多个候选模型中选择最佳者
- **性能保证**：确保模型在实际应用中表现良好

### 2. 训练/验证/测试三分法
```
训练集 → 训练模型
验证集 → 调参、选择模型
测试集 → 最终评估
```

### 3. 核心评估指标

#### 混淆矩阵基础
```
          预测
实际    正类    负类
正类    TP      FN
负类    FP      TN
```

#### 四大核心指标
1. **准确率 (Accuracy)** = (TP+TN)/(TP+FP+TN+FN)
2. **精确率 (Precision)** = TP/(TP+FP)  -> 预测为正的中有多少是真正
3. **召回率 (Recall)** = TP/(TP+FN)     -> 真正的有多少被找出来
4. **F1分数** = 2×(Precision×Recall)/(Precision+Recall)

## 直觉理解 / 类比理解

### 医生诊断类比
- **准确率**：所有诊断中有多少是对的
- **精确率**：诊断为"生病"的患者中有多少真的生病
- **召回率**：真正生病的患者中有多少被诊断出来

### 招聘面试类比
- **精确率**：面试通过的人中有多少是真合适的
- **召回率**：所有合适的人中有多少来面试了

## 关键机制或关键公式

### 1. 交叉验证
```python
# K折交叉验证
from sklearn.model_selection import cross_val_score
scores = cross_val_score(model, X, y, cv=5)
print(f"平均准确率: {scores.mean():.3f} ± {scores.std():.3f}")
```

### 2. ROC曲线与AUC
```python
from sklearn.metrics import roc_auc_score, roc_curve

fpr, tpr, thresholds = roc_curve(y_true, y_pred_proba)
auc_score = roc_auc_score(y_true, y_pred_proba)

# 绘制ROC曲线
plt.plot(fpr, tpr, label=f'AUC = {auc_score:.3f}')
plt.plot([0,1], [0,1], '--', label='Random')
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
```

### 3. 学习曲线诊断
```python
from sklearn.model_selection import learning_curve

train_sizes, train_scores, val_scores = learning_curve(
    model, X, y, cv=5, train_sizes=np.linspace(0.1, 1.0, 10)
)
```

## 最小示例

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LogisticRegression
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.metrics import classification_report, confusion_matrix

# 生成示例数据
np.random.seed(42)
X = np.random.randn(100, 2)
y = (X[:, 0] + X[:, 1] > 0).astype(int)

# 分割数据
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# 训练模型
model = LogisticRegression()
model.fit(X_train, y_train)

# 基本评估
y_pred = model.predict(X_test)
print("分类报告：")
print(classification_report(y_test, y_pred))

# 交叉验证
cv_scores = cross_val_score(model, X, y, cv=5)
print(f"交叉验证准确率: {cv_scores.mean():.3f} ± {cv_scores.std():.3f}")

# 混淆矩阵
cm = confusion_matrix(y_test, y_pred)
print("混淆矩阵：")
print(cm)
```

## 今天学完后我应该记住什么

1. **评估框架**：训练/验证/测试三分法是评估基础
2. **指标选择**：准确率不够用，要根据具体问题选择精确率、召回率等
3. **交叉验证**：5折或10折交叉验证比单一分割更稳健
4. **过拟合检测**：通过训练集和验证集性能差距判断
5. **ROC-AUC**：衡量分类器整体性能的重要指标
