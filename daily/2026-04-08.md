## 今日学习主题
**逻辑回归：从连续预测到概率分类**

## 为什么今天学这个？
逻辑回归是机器学习的第二个核心算法，也是分类任务的基石。昨天学习了线性回归预测连续值，今天学习如何用它来做分类问题。这是从"回归"到"分类"的重要转变，也是理解后续所有分类算法的基础。

## 这个知识点在 AI 演进中的位置
- **经典算法阶段**：逻辑回归（19世纪中叶概念，现代应用在1950s-1960s）
- **承前启后**：在线性回归基础上发展，为神经网络和深度学习奠定基础
- **现代应用**：至今仍然是医疗诊断、信用评估、垃圾邮件检测等场景的基准算法

## 核心概念解释

### 1. 分类 vs 回归的区别
- **回归**：预测连续值（房价、温度、股价）
- **分类**：预测离散标签（是/否、猫/狗、A/B/C）

### 2. 逻辑回归的核心思想
在线性回归基础上增加 **sigmoid 函数**，将连续值转换为 [0,1] 区间的概率：
```
z = kx + b  （线性部分）
p = 1/(1+e^(-z))  （sigmoid激活）
```

### 3. 决策边界
设定阈值（通常是0.5）：
- p ≥ 0.5 → 正类（1）
- p < 0.5 → 负类（0）

## 直觉理解 / 类比理解

### 风险评估师
想象你是银行风控员：
- **线性回归**：直接预测"用户会欠款多少元"（具体数字）
- **逻辑回归**：预测"用户会欠款的概率"（0-1之间的概率）
- **最终决策**：概率 ≥ 50% → 拒绝贷款

### 温度计类比
- 线性回归：精确到"36.5°C"
- 逻辑回归：告诉你"发烧的概率是75%"

## 关键机制或关键公式

### 1. Sigmoid 函数
```
σ(z) = 1/(1+e^(-z))
```
- 输入：任意实数 z
- 输出：0到1之间的概率值
- 特点：S形曲线，中心在(0,0.5)

### 2. 损失函数：交叉熵
```
L = -[y·log(p) + (1-y)·log(1-p)]
```
- y：真实标签（0或1）
- p：预测概率
- 特点：对错误预测惩罚更大

### 3. 梯度下降更新
```
k = k - α·∂L/∂k
b = b - α·∂L/∂b
```

## 最小示例

```python
import numpy as np
import matplotlib.pyplot as plt

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

# 线性部分
z = 2*x + 1

# 激活函数
p = sigmoid(z)

# 预测
prediction = 1 if p >= 0.5 else 0

# 可视化
plt.figure(figsize=(12, 4))
plt.subplot(1, 2, 1)
plt.plot(z, p)
plt.title('Sigmoid Function')

plt.subplot(1, 2, 2)
plt.scatter(x, y, c=prediction, cmap='RdYlBu')
plt.title('Classification Results')
```
