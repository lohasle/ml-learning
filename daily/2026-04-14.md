# 2026年4月14日 - 第7课：SVM 支持向量机

## 今日学习主题

**掌握支持向量机的核心思想——「在两类之间找最宽的那条路」，理解最大间隔、支持向量、核技巧**

## 为什么学这个？

上一课我们学了随机森林与集成方法——通过组合多个「弱学习器」来提升性能。SVM 走的是完全不同的路线：它不组队，而是靠**几何直觉**——找一条离两边都最远的决策边界。

SVM 是机器学习历史上最重要的算法之一，在深度学习崛起之前，它几乎垄断了中小规模数据集的分类任务。即使在今天，SVM 在文本分类、生物信息学、图像识别等场景中仍然是强有力的基线模型。

**在 AI 演进史中的位置**：SVM（1995年 Vapnik 提出）是连接经典统计学习理论与现代机器学习的桥梁。它引入的**核技巧**思想，后来深刻影响了核方法、高斯过程，甚至深度学习中的核回归视角。

## 核心概念解释

### 1. 最大间隔分类器

- **直觉**：两个阵营之间修一条最宽的马路，马路中线就是决策边界
- **数学**：最小化 $\frac{1}{2}\|w\|^2$，使得 $y_i(w \cdot x_i + b) \geq 1$
- **关键**：间隔宽度 = $\frac{2}{\|w\|}$，$\|w\|$ 越小间隔越宽

### 2. 支持向量

- **直觉**：离马路最近的那些点——它们「撑」起了整个决策边界
- **关键洞察**：删掉一个普通点，模型不变；删掉一个支持向量，边界会移动
- **代码直觉**：`svm.support_vectors_` 返回的就是这些关键点

### 3. 软间隔与 C 参数

- **直觉**：允许少量点「犯规」（落在马路里），但犯规要付出代价
- **C 参数**：
  - C 大 → 严格，不轻易容忍误分类 → 间隔窄 → 可能过拟合
  - C 小 → 宽容，允许更多误分类 → 间隔宽 → 可能欠拟合
- **与第 4 课（过拟合与正则化）的联系**：C 本质上就是正则化参数

### 4. 核技巧

- **直觉**：数据在低维线性不可分？把它「弹」到高维，就能分了
- **魔法**：不需要真的计算高维映射，只需用核函数计算「内积」
- **类比**：你不需要知道 3D 空间的坐标，只需知道点之间的「相似度」

## 关键公式直觉

| 概念 | 公式 | 直觉解释 |
|------|------|----------|
| 优化目标 | $\min \frac{1}{2}\|w\|^2$ | 让马路尽量宽 |
| 间隔宽度 | $\frac{2}{\|w\|}$ | w 越短，路越宽 |
| 软间隔损失 | $C \sum \xi_i + \frac{1}{2}\|w\|^2$ | 间隔宽度 vs 分类错误的平衡 |
| RBF 核 | $K(x,z) = e^{-\gamma\|x-z\|^2}$ | 两点越近，相似度越高 |
| 决策函数 | $f(x) = \sum_{i} \alpha_i y_i K(x_i, x) + b$ | 只依赖支持向量的加权投票 |

## 常见问题

**Q: SVM 和逻辑回归有什么区别？**
A: 核心区别在于优化目标。逻辑回归最小化分类误差（所有点都参与），SVM 最大化间隔（只关注边界附近的点）。SVM 通常在中小数据集上效果更好，逻辑回归在大数据集上更稳定。

**Q: 什么时候用线性核，什么时候用 RBF 核？**
A: 经验法则——如果特征数远大于样本数（如文本分类），线性核就够用且更快；如果样本数大于特征数，优先试 RBF 核。

**Q: gamma 和 C 怎么调？**
A: 推荐用 `GridSearchCV` 搜索 `C=[0.1, 1, 10, 100]` × `gamma=[0.01, 0.1, 1, 10]` 的组合。实践中这两个参数有交互效应，需要一起调。

**Q: SVM 能做回归吗？**
A: 能！`SVR`（Support Vector Regression）用类似的思路，找一条让误差在 ε 范围内的最「平」的线。

## 本课总结

1. **SVM 的核心是最大间隔**——不是「分对就行」，而是「分得最稳」
2. **支持向量是模型的关键**——只有少数点决定了决策边界
3. **核技巧是 SVM 的魔法**——隐式映射到高维，让非线性变线性
4. **C 和 gamma 是两个关键旋钮**——控制分类严格程度和每个点的影响范围
5. **SVM 对特征缩放敏感**——用之前必须标准化

## 关键历史节点

- **1963年**：Vapnik 和 Chervonenkis 提出 VC 维理论（SVM 的理论基础）
- **1992年**：Boser, Guyon, Vapnik 引入核技巧到 SVM
- **1995年**：Cortes 和 Vapnik 发表软间隔 SVM 论文
- **1998-2012年**：SVM 在文本分类、生物信息学等领域占据主导地位
- **2012年后**：深度学习崛起，但 SVM 在中小数据集上仍然是强基线
