# 2026年4月12日 - 第6课：随机森林与集成方法

## 今日学习主题
**从一棵树到一片森林——集成学习如何让弱学习器变成强学习器**

## 为什么学这个？
上一课我们学了决策树——直观、可解释，但容易过拟合。今天学随机森林，就是决策树的「升级版」：**用一群决策树投票，比单棵树更准、更稳**。

集成学习是现代 ML 最成功的思想之一，Kaggle 比赛中大部分冠军方案都用到了集成方法。理解 Bagging 和 Boosting，是进入高级 ML 的关键门槛。

## 核心概念解释

### 1. 集成学习（Ensemble Learning）
- **直觉**：三个臭皮匠顶个诸葛亮
- **数学基础**：大数定律。假设每个弱分类器正确率 $p > 0.5$，$N$ 个独立分类器投票，整体正确率随 $N$ 增大趋近 1
- **关键前提**：个体学习器的错误要尽量「不相关」——差异性越大，集成效果越好

### 2. Bagging（Bootstrap Aggregating）
- **流程**：从训练集有放回抽样 → 生成多个子数据集 → 分别训练模型 → 投票/平均
- **为什么有效**：每棵树看到不同数据子集，错误不相关，投票抵消个体错误
- **降低什么**：主要降低**方差**（variance）
- **代表算法**：BaggingClassifier、随机森林

### 3. 随机森林（Random Forest）
- 在 Bagging 基础上加一层随机性：每次分裂只从**随机子集的特征**中选择
- 默认每次分裂考虑 $\sqrt{d}$ 个特征（$d$ 是总特征数）
- **为什么需要特征随机**？如果某个特征特别强，所有树都优先用它 → 树之间高度相关 → 集成效果打折
- **附加好处**：自动提供特征重要性排序

### 4. Boosting（提升方法）
- **核心思想**：串行训练，每一棵新树重点关注前面所有树犯错的样本
- **AdaBoost**：提高被错分样本的权重，下一棵树更关注这些「难题」
- **GBDT**：每棵新树拟合前面所有树的残差（预测误差）
- **降低什么**：主要降低**偏差**（bias）
- **代表算法**：AdaBoost、GBDT、XGBoost、LightGBM

## 关键公式直觉

| 概念 | 公式/机制 | 直觉解释 |
|------|-----------|----------|
| Bootstrap | 有放回抽样 $N$ 次 | 每棵树看到的训练集有差异，约 63% 的不同样本 |
| 投票集成 | $\hat{y} = \text{majority}(h_1(x), ..., h_N(x))$ | 多数人说了算 |
| 特征随机 | 每次分裂只看 $\sqrt{d}$ 个特征 | 别让所有树都盯着同一个强特征 |
| AdaBoost 样本权重 | 错分样本权重加大 | 难题重点攻克 |
| GBDT 残差拟合 | $F_{m}(x) = F_{m-1}(x) + h_m(x)$ | 每棵新树补上前面没做好的部分 |

## Bagging vs Boosting 选择指南

| 场景 | 推荐 | 原因 |
|------|------|------|
| 数据有噪声 | Bagging / 随机森林 | 对噪声不敏感 |
| 需要最高精度 | Boosting (XGBoost) | 偏差和方差都能降 |
| 数据量很大 | 随机森林 | 可并行训练，速度快 |
| 快速出基线 | 随机森林 | 调参简单，默认参数就不错 |
| 比赛冲分 | Boosting + Bagging 混合 | 互补效果最好 |

## 常见问题

**Q: 随机森林不容易过拟合，是真的吗？**
A: 增加树的数量不会过拟合（因为只是投票），但单棵树深度太深仍可能过拟合。关键参数：`max_depth`、`min_samples_leaf`。

**Q: XGBoost 为什么比 sklearn 的 GBDT 强？**
A: XGBoost 加了正则化（防止过拟合）、列采样（类似随机森林的特征随机）、并行化、处理缺失值等工程优化，是 GBDT 的工业级实现。

**Q: 集成学习能用在深度学习吗？**
A: 可以！Dropout 本质上就是一种集成（训练时随机丢弃神经元 = 训练多个子网络），推理时取平均。

## 本课总结
1. 集成学习的核心：组合多个弱学习器 → 强学习器，数学基础是大数定律
2. Bagging 通过并行训练 + 投票降低**方差**，代表是随机森林
3. Boosting 通过串行训练 + 关注错误降低**偏差**，代表是 GBDT、XGBoost
4. 随机森林在 Bagging 基础上加特征随机，进一步增加树之间的差异性
5. 特征重要性是随机森林的附加红利，能帮助理解数据
