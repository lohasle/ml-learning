# 2026年4月25日 - 第17课：预训练语言模型——BERT 与 GPT

## 今日学习主题
**理解「预训练 + 微调」范式，以及 BERT 和 GPT 如何从 Transformer 架构分道扬镳，分别引爆了 NLP 理解和文本生成革命。**

## 为什么今天学这个？
上一课学了 Transformer——Self-Attention、多头注意力、位置编码。但 Transformer 只是骨架，真正让它在 NLP 领域大杀四方的是两个具体实现：

- **BERT（2018.10）**：Google 出品，用 Transformer Encoder，双向看上下文
- **GPT-1（2018.06）**：OpenAI 出品，用 Transformer Decoder，从左到右生成

这两个模型几乎同时出现，却走上了完全不同的路线，最终共同定义了现代 AI 的格局。

## 在 AI 演进中的位置
```
RNN/LSTM → Attention → Transformer → ★ BERT/GPT ★ → GPT-3 → ChatGPT → 大模型时代
```
这是从「任务专用模型」到「通用预训练模型」的关键转折点。

## 核心概念

### 1. 预训练 + 微调范式

**以前的做法：** 每个任务从零训练 → 需要大量标注数据
**现在的做法：** 先在海量无标注文本上预训练（学会语言规律）→ 再在下游任务上微调

**类比：** 传统方式像从零培养一个律师；预训练方式像先读完法学百科全书，再实习。

### 2. BERT：双向理解的代表

- **架构：** Transformer Encoder（6-24 层）
- **训练目标：MLM（Masked Language Model）**——随机遮住 15% 的词，让模型根据左右上下文猜
- **类比：** 完形填空
- **额外目标：NSP（Next Sentence Prediction）**——判断两个句子是否相邻
- **代表作：** BERT-Base（110M 参数）、BERT-Large（340M 参数）

### 3. GPT：自回归生成的代表

- **架构：** Transformer Decoder（12-96 层）
- **训练目标：CLM（Causal Language Model）**——根据左边所有词预测下一个词
- **类比：** 打字预测（手机输入法的超级加强版）
- **演进路线：** GPT-1（117M）→ GPT-2（1.5B）→ GPT-3（175B）

### 4. 为什么走向不同结局？

| 维度 | BERT | GPT |
|------|------|-----|
| 注意力方向 | 双向（看左右） | 单向（只看左边） |
| 训练目标 | MLM（完形填空） | CLM（预测下一个） |
| 擅长 | 理解、分类、问答 | 生成、对话、创作 |
| 规模上限 | ~340M 后提升有限 | 持续 Scaling Law |
| 后续影响 | 开启了 NLP 预训练时代 | 开启了生成式 AI 时代 |

**GPT 路线最终胜出**，核心原因是：生成能力 + 规模扩展 + In-Context Learning 的涌现。

## 关键公式直觉

| 概念 | 公式/机制 | 直觉解释 |
|------|----------|----------|
| MLM 损失 | L = -Σ log P(masked_token \| context) | 完形填空：看上下文猜被遮的词 |
| CLM 损失 | L = -Σ log P(x_t \| x_1...x_{t-1}) | 只看前面，猜下一个词 |
| Self-Attention | Attention(Q,K,V) = softmax(QK^T/√d_k)V | 上一课学过，BERT/GPT 的核心计算 |
| Scaling Law | Performance ∝ C^α (C=计算量) | 规模越大，效果越好（GPT 的哲学） |

## 常见问题

**Q: BERT 和 GPT 能互相替代吗？**
A: 不完全能。BERT 在理解任务（分类、NER）上更高效；GPT 在生成任务上更强。但 GPT 规模够大后，也能做理解任务（这就是 In-Context Learning）。

**Q: 预训练的数据从哪来？**
A: BERT 用了 BooksCorpus + 英文维基百科（约 16GB 文本）。GPT-2 用 WebText（约 40GB）。GPT-3 用了 CommonCrawl 等混合数据（约 570GB）。数据规模和多样性是关键。

**Q: 为什么 BERT 不继续 Scaling？**
A: 双向注意力对计算资源的要求随序列长度二次增长，且 MLM 目标的收益在更大规模时递减。GPT 的自回归目标更适合持续扩展。

## 本课总结
1. 预训练 + 微调范式是现代 NLP 的基石——先学语言规律，再适配具体任务
2. BERT 用双向 Encoder + MLM，擅长理解；GPT 用单向 Decoder + CLM，擅长生成
3. GPT 路线因 Scaling Law 和涌现能力最终成为大模型的主流方向
4. 这两个模型的同时出现（2018 年）标志着 AI 进入了「预训练模型时代」
