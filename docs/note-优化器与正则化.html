核心概念

优化器

优化器决定了「拿到梯度后怎么更新参数」：

优化器 核心机制 优势 劣势
SGD θ ← θ - η·∇L 简单，理论清晰 收敛慢，需调 lr
Momentum 累积历史梯度方向 加速收敛，冲出局部极小 需调 β
Adam 一阶矩 + 二阶矩自适应 快速稳定，少调参 可能过拟合

正则化

方法 机制 直觉
L2 正则化 损失函数加 λ·Σw² 不让权重太大
Dropout 训练时随机丢弃神经元 强迫冗余表示
BatchNorm 归一化每层输出 稳定训练 + 轻微正则化
早停 (Early Stopping) 验证 loss 不降就停止 不给模型过拟合的机会

关键公式

  • Momentum: v_t = β·v_{t-1} + (1-β)·∇L; θ = θ - η·v_t
  • Adam: m_t = β₁·m_{t-1} + (1-β₁)·∇L; v_t = β₂·v_{t-1} + (1-β₂)·(∇L)²; θ = θ - η·m̂_t/(√v̂_t + ε)
  • L2: L_total = L_loss + (λ/2)·Σ(w_i²)
