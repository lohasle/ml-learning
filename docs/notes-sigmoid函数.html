Sigmoid 函数知识卡片

🎯 基础概念

1. 定义

Sigmoid 函数，也称为 S 型函数，是一个将任意实数映射到 (0, 1) 区间的数学函数。

2. 数学表达式

σ(z) = 1 / (1 + e^(-z))

3. 别名

  • Logistic 函数
  • S 型激活函数
  • Softmax 函数的单变量版本

📊 函数特性

1. 域与值域

  • 定义域：z ∈ (-∞, +∞)
  • 值域：σ(z) ∈ (0, 1)
  • 边界值：
  • lim(z→-∞) σ(z) = 0
  • lim(z→+∞) σ(z) = 1

2. 关键特性

  • 单调递增：导数始终为正
  • 中心对称：σ(-z) = 1 - σ(z)
  • S型曲线：缓入-急变-缓出
  • 在 z=0 处值为 0.5

3. 导数性质

σ'(z) = σ(z) · (1 - σ(z))
  • 在 z=0 处导数最大：σ'(0) = 0.25
  • 随 |z| 增大，导数趋近于 0

1. 基本形状

2. 关键点

  • 拐点：z = 0, σ(z) = 0.5
  • 最陡峭处：z = 0（导数最大）
  • 饱和区：|z| > 3，变化缓慢

🔧 应用场景

1. 概率建模

  • 二分类概率：将线性组合转为概率
  • 信念强度：表示事件发生的可能性
  • 激活值：神经元输出的概率解释

2. 激活函数

  • 神经网络：隐藏层激活函数
  • 输出层：二分类问题的概率输出
  • 门控机制：神经网络中的控制信号

3. 时间序列分析

  • 增长模型：描述 S 型增长过程
  • 人口模型：Verhulst 方程的基础
  • 扩散过程：技术扩散的建模

⚡ 计算特性

1. 数值稳定性

  • 大负数：e^(-z) → ∞，σ(z) ≈ 0
  • 大正数：e^(-z) → 0，σ(z) ≈ 1
  • 溢出问题：直接计算可能导致数值溢出

2. 优化计算

避免数值溢出：
if z ≥ 0:
    σ(z) = 1 / (1 + e^(-z))
else:
    σ(z) = e^z / (1 + e^z)

3. 导数计算

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def sigmoid_derivative(z):
    s = sigmoid(z)
    return s * (1 - s)  # 利用函数值计算导数

🔄 函数变种

1. 双曲正切函数

tanh(z) = (e^z - e^(-z)) / (e^z + e^(-z)) = 2σ(2z) - 1
  • 值域：(-1, 1)
  • 中心：0
  • 用于零中心化的激活

2. Softplus 函数

softplus(z) = ln(1 + e^z)
  • Sigmoid 函数的积分
  • 更平滑的 ReLU 变种

3. Swish 函数

swish(z) = z · σ(z)
  • 自门控激活函数
  • 在深度学习中表现优异

📐 数学性质

1. 不动点

  • σ(z) = z 的解约为 z ≈ 0.1587
  • 在深度学习中用于初始化

2. 泰勒展开

在 z=0 处展开：

σ(z) ≈ 0.5 + 0.25z - 0.0625z² + 0.0208z³ - ...

3. 对数导数

d(ln(σ(z))) / dz = 1 - σ(z)
d(ln(1-σ(z))) / dz = -σ(z)

⚠️ 局限性

1. 梯度消失

  • 当 |z| 很大时，σ'(z) ≈ 0
  • 影响深度神经网络的训练

2. 零中心输出

  • σ(z) ∈ (0, 1)，均值不为0
  • 可能影响后续层的训练

3. 计算成本

  • 指数计算相对复杂
  • 可能有更高效的替代函数

🚀 替代方案

1. ReLU 函数

ReLU(z) = max(0, z)
  • 解决梯度消失问题
  • 计算简单，收敛快

2. Leaky ReLU

LeakyReLU(z) = max(αz, z), α ≈ 0.01
  • 解决 ReLU 死亡问题
  • 保持零中心特性

3. ELU 函数

ELU(z) = { z, if z ≥ 0; α(e^z - 1), if z < 0 }
  • 平滑负值区域
  • 缓解梯度消失

📚 最佳实践

1. 使用场景选择

  • 二分类输出：首选 Sigmoid
  • 隐藏层：考虑 ReLU 或其变种
  • 需要概率解释：必须使用 Sigmoid

2. 参数初始化

  • 初始 z 值应在 sigmoid 的线性区域
  • 避免初始值在饱和区

3. 数值稳定性

  • 使用对数概率空间计算
  • 避免直接计算大指数
