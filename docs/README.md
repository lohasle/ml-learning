# GitHub Pages 目录

本目录为 ml-learning 仓库的 GitHub Pages 站点源文件。

## 页面结构

- `index.html` — 首页（学习总览与快速导航）
- `roadmap.html` — 学习路线图
- `daily.html` — 每日学习记录索引
- `weekly.html` — 每周复盘索引
- `notes.html` — 知识卡片索引
- `history.html` — AI 技术演进历史
- `styles.css` — 共享样式

## 更新方式

日常学习推送时，Agent 会自动更新仓库目录（daily/、weekly/、notes/ 等）和本 Pages 目录中的对应页面。
