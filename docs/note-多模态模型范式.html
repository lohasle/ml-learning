多模态模型范式

🎯 核心概念

1. 基础定义

  • 类型：多模态学习 / 跨模态对齐
  • 适用场景：图文匹配、零样本分类、视觉问答、图片描述、多模态生成
  • 一句话定义：让 AI 同时理解和关联不同类型的数据（文本、图像、音频、视频）

2. 三种核心范式

共享嵌入空间（CLIP）

  • 双编码器架构：图像编码器 + 文本编码器
  • 对比学习（InfoNCE 损失）让配对的图文在向量空间中靠近
  • 训练后可以做零样本分类：任意图片 vs 任意文字描述

跨模态注意力（LLaVA / Flamingo）

  • 在 LLM 中插入视觉交叉注意力层
  • 文本 token 通过 Attention 机制「查询」视觉特征
  • 可以冻结 LLM，只训练适配器，成本低

统一 Token 化（GPT-4V / Gemini）

  • 所有模态统一编码为 token
  • 端到端联合训练，共享参数
  • 当前最强大但训练成本最高的范式

3. 数学原理

公式 含义 直觉
$\text{CLIP Loss} = -\log \frac{\exp(v_i \cdot t_i / \tau)}{\sum_j \exp(v_i \cdot t_j / \tau)}$ 让配对图文互相「认出对方」 派对上让认识的人坐一起
$\cos(v, t) = \frac{v \cdot t}{|v| |t|}$ 余弦相似度 两个向量指向同一方向的程度
$\text{CrossAttn}(Q_t, K_v, V_v)$ 文本查询视觉 你问「猫在哪里」，模型看图回答
$\text{Image} \to [p_1, ..., p_N]$ 图像 Token 化 把图片切成小块变成「视觉词」

📊 关键里程碑

年份 模型 范式 意义
2021 CLIP 共享嵌入 首次证明对比学习可以实现强大零样本能力
2022 Flamingo 跨模态注意力 Few-shot 视觉推理
2023 LLaVA 跨模态注意力 开源视觉语言模型标杆
2023 GPT-4V 统一 Token 商业级多模态推理
2023 Gemini 统一 Token 原生多模态设计
2024 GPT-4o 统一 Token 文本+图像+音频统一处理

🔗 关联概念

  • 前置知识：CNN（图像特征提取）、Transformer（序列建模）、Attention（注意力机制）、LLM（语言模型基础）
  • 后续应用：AI 系统（多模态 Agent）、训练工程（大规模多模态训练）、部署（多模态推理优化）
  • 同类对比：
  • CLIP vs 传统图像分类：零样本 vs 固定类别
  • LLaVA vs GPT-4V：适配器方案 vs 原生多模态
  • 多模态 vs 单模态：跨模态关联 vs 单一领域

💡 记忆技巧

多模态 = 给 AI 装上「眼睛和耳朵」。CLIP 是「配对游戏」（图文匹配），LLaVA 是「参考图写文章」（一边看图一边写），GPT-4V 是「天生全才」（所有感官融为一体）。
