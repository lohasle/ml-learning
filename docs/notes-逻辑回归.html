逻辑回归知识卡片

🎯 核心概念

1. 基础定义

  • 类型：分类算法
  • 适用场景：二分类问题（是/否、真/假、成功/失败）
  • 输出：概率值 [0, 1]

2. 与线性回归的区别

特性 线性回归 逻辑回归
输出类型 连续值 概率值
问题类型 回归 分类
激活函数 无 sigmoid
损失函数 均方误差 交叉熵

📐 数学原理

1. Sigmoid 函数

σ(z) = 1 / (1 + e^(-z))
  • 输入：任意实数 z
  • 输出：(0, 1) 区间的概率
  • 特性：S型曲线，中心对称

2. 线性组合

z = k₁x₁ + k₂x₂ + ... + kₙxₙ + b
  • kᵢ：特征权重
  • xᵢ：特征值
  • b：截距项

3. 概率计算

p = σ(z) = 1 / (1 + e^(-(k₁x₁ + k₂x₂ + ... + b)))

4. 决策规则

预测类别 = 1, 如果 p ≥ threshold
预测类别 = 0, 如果 p < threshold
  • 默认 threshold = 0.5
  • 决策边界：z = 0

🔧 实现要点

1. scikit-learn 实现

from sklearn.linear_model import LogisticRegression

# 创建模型
model = LogisticRegression()

# 训练
model.fit(X_train, y_train)

# 预测
predictions = model.predict(X_test)
probabilities = model.predict_proba(X_test)[:, 1]

2. 关键参数

  • C：正则化强度（越小正则化越强）
  • class_weight：类别权重（处理不平衡数据）
  • solver：优化算法（liblinear, lbfgs等）
  • max_iter：最大迭代次数

3. 决策边界

  • 线性边界：z = 0
  • 公式：k₁x₁ + k₂x₂ + ... + b = 0
  • 可视化：分类区域的分界线

📊 评估指标

1. 基础指标

  • 准确率：正确预测的比例
  • 精确率：预测为正例中真正为正例的比例
  • 召回率：真正为正例中被预测出来的比例
  • F1分数：精确率和召回率的调和平均

2. 混淆矩阵

           预测
实际   负例  正例
      ┌──────┬──────┐
负例  │ TN   │ FP   │
      ├──────┼──────┤
正例  │ FN   │ TP   │
      └──────┴──────┘

🎨 应用场景

1. 商业应用

  • 信用评估：判断客户违约风险
  • 营销预测：客户购买意愿分析
  • 欺诈检测：异常交易识别

2. 医疗诊断

  • 疾病预测：患病风险评估
  • 诊断辅助：症状分析
  • 预后判断：治疗效果预测

3. 自然语言处理

  • 情感分析：正面/负面情感分类
  • 垃圾邮件检测：正常/垃圾邮件分类
  • 文本分类：主题分类

⚠️ 局限性

1. 技术局限

  • 只能处理线性可分问题
  • 对特征缩放敏感
  • 多类别需要扩展（One-vs-Rest）

2. 数据要求

  • 需要足够的训练样本
  • 特征之间不应高度相关
  • 类别不平衡需要特殊处理

3. 应用限制

  • 无法处理非线性关系
  • 对异常值较敏感
  • 解释性相对有限

🚀 进阶方向

1. 多分类扩展

  • One-vs-Rest (OvR)
  • One-vs-One (OvO)
  • Softmax 回归

2. 正则化方法

  • L1 正则化：特征选择
  • L2 正则化：防止过拟合
  • ElasticNet：L1+L2组合

3. 非线性扩展

  • 核方法
  • 多项式特征
  • 神经网络

学习目标：掌握逻辑回归的基本原理、实现方法和应用场景，能够独立应用于二分类问题。
