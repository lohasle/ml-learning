大模型微调（Fine-tuning）

概述

微调是让预训练模型适配特定任务的关键技术。从全量微调到参数高效微调（PEFT），核心目标都是用最少的资源获得最大的领域适配效果。

核心方法对比

方法 可训练参数 显存 效果 适用场景
Prompt Engineering 0 最低 有限 快速验证
Full Fine-tuning 100% 最高 最好 数据充足、追求极致
LoRA ~0.1% 中等 接近 Full FT 工业界首选
QLoRA ~0.1% 较低 接近 LoRA 资源受限
Adapter ~1-5% 中等 好 多任务切换

关键公式

LoRA 核心公式

W' = W + (α/r) × A × B
  • W: 原始权重 (d×d)，冻结不更新
  • A: 下投影矩阵 (d×r)
  • B: 上投影矩阵 (r×d)
  • r: 秩（通常 4~64）
  • α: 缩放因子

参数量对比

  • 原始: d × d
  • LoRA: 2 × d × r
  • 压缩比: d / (2r)
