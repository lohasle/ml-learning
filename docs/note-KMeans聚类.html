KMeans 聚类

概述

无监督学习中最经典的聚类算法。不需要标签，通过迭代「分配→更新」自动将数据分成 K 组。

核心机制

算法步骤

  1. 随机选 K 个初始中心点
  2. 将每个样本分配到最近的中心
  3. 重新计算每个簇的中心（取均值）
  4. 重复直到收敛

关键公式

公式 含义
d(x,c) = ‖x-c‖₂ 样本到中心距离
Inertia = ΣΣ‖x-cᵢ‖² 簇内平方和（优化目标）
cᵢ = mean(x∈簇i) 中心更新规则

评估方法

  • 肘部法则：Inertia vs K 曲线的拐点
  • 轮廓系数：[-1, 1]，越大越好

局限性

  • 只能发现球形簇
  • 需要预设 K
  • 对初始化敏感（k-means++ 缓解）

关联概念

  • [[KNN]]：KNN 找最近邻，KMeans 找最近中心
  • [[PCA]]：常先降维再聚类
  • [[DBSCAN]]：能处理非球形簇的替代方案

历史节点

  • 1957：Lloyd 提出（贝尔实验室）
  • 至今仍是数据探索首选基线
