# KNN 与朴素贝叶斯

## 概述
KNN 和朴素贝叶斯是两种经典的分类算法，分别代表「基于实例」和「基于概率」的分类哲学。

## KNN 核心机制
- **距离度量**：d(x,y) = √Σ(xi-yi)²
- **K 值选择**：经验值 √n，用交叉验证确定最佳 K
- **加权投票**：近邻权重更大可提升效果

## 朴素贝叶斯核心机制
- **贝叶斯定理**：P(C|X) ∝ P(X|C)·P(C)
- **条件独立假设**：P(X|C) = Π P(xi|C)
- **三种变体**：GaussianNB（连续）、MultinomialNB（计数）、BernoulliNB（二值）

## 关联概念
| 概念 | 关联 |
|------|------|
| 距离度量 | → 下一课 KMeans 也依赖距离 |
| 贝叶斯定理 | → 后续贝叶斯网络、概率图模型 |
| 懒惰学习 | vs 急切学习（SVM、决策树等） |

## 相关历史
- 1951：KNN 雏形（Fix & Hodges）
- 1960s：朴素贝叶斯用于文本分类
- 2002：Paul Graham「A Plan for Spam」推广贝叶斯邮件过滤
