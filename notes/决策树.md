# 决策树

> 第5课知识卡片 | 2026-04-11

## 一句话概括

决策树通过递归选择最优特征和分裂点，将数据逐步划分为越来越纯的子集，形成一棵 if-else 规则树。

## 核心要素

### 树的结构
```
根节点（全部数据）
  ├── 内部节点（特征判断）
  │     ├── 左子树（条件成立）
  │     └── 右子树（条件不成立）
  └── 叶子节点（预测结果）
```

### 分裂准则

| 准则 | 公式 | 使用者 |
|------|------|--------|
| 信息熵 | H = -Σ p_i log₂(p_i) | ID3 |
| 信息增益率 | Gain Ratio = Info Gain / Intrinsic Info | C4.5 |
| 基尼不纯度 | Gini = 1 - Σ p_i² | CART（sklearn 默认）|

### 关键超参数

| 参数 | 作用 | 建议 |
|------|------|------|
| `max_depth` | 树的最大深度 | **最重要**，从 3-5 开始试 |
| `min_samples_split` | 分裂所需最小样本数 | 防止过细分裂 |
| `min_samples_leaf` | 叶子最小样本数 | 保证叶子有统计意义 |
| `criterion` | 分裂准则 | gini（默认）或 entropy |

## 优缺点

### ✅ 优点
- **高度可解释**：决策路径可视化，非技术人员也能理解
- **无需特征缩放**：不像线性模型需要对特征做标准化
- **处理混合数据**：数值特征和类别特征都能处理
- **特征重要性**：训练后自动给出各特征的重要性排名

### ❌ 缺点
- **容易过拟合**：不加限制会过度生长
- **不稳定**：数据稍有变化，树结构可能大变
- **偏向多值特征**：信息增益准则容易偏向取值多的特征（C4.5 通过增益率修正）
- **边界是轴对齐的**：决策边界只能水平/垂直，处理对角分布的数据效果差

## 与前后知识的关系

```
线性回归/逻辑回归（线性模型，全局一个公式）
        ↓ 局限：只能处理线性关系
决策树（非线性，分段规则，局部决策）
        ↓ 局限：单棵树不稳定
随机森林 / 梯度提升树（多棵树集成，稳健）
        ↓ 启发：模型组合的思想
神经网络（另一种处理非线性的方式）
```
