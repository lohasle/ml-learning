# SVM 支持向量机

## 🎯 核心概念

### 1. 基础定义
- **类型**：分类（SVC）/ 回归（SVR）
- **适用场景**：中小规模数据集、高维特征、需要精确决策边界
- **一句话定义**：在两类之间找一条离两边最近点都最远的决策边界

### 2. 数学原理

**最大间隔优化**：
$$\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w \cdot x_i + b) \geq 1$$

**软间隔松弛**：
$$\min_{w,b,\xi} \frac{1}{2}\|w\|^2 + C\sum_{i}\xi_i \quad \text{s.t.} \quad y_i(w \cdot x_i + b) \geq 1 - \xi_i$$

**直觉**：
- $\frac{1}{2}\|w\|^2$：让间隔尽量宽
- $C\sum\xi_i$：允许一些点越界，但要付出代价
- 这就是**正则化**思想——和第 4 课完全一致

### 3. 核技巧

$$K(x, z) = \phi(x) \cdot \phi(z)$$

核函数的本质：**不显式计算高维映射 $\phi(x)$，直接计算内积**。

常用核函数：
- **线性核**：$K(x,z) = x \cdot z$ — 文本、高维稀疏数据
- **多项式核**：$K(x,z) = (x \cdot z + c)^d$ — 特征有交互
- **RBF 核**：$K(x,z) = e^{-\gamma\|x-z\|^2}$ — 通用默认选择

### 4. 实践要点
- **特征必须标准化**：SVM 对尺度敏感
- **C 参数**：正则化强度的倒数，大 C → 严格，小 C → 宽容
- **gamma 参数**（RBF）：控制单个支持向量的影响半径
- **调参建议**：GridSearchCV 搜索 C × gamma 组合

## 📊 关键公式

| 公式 | 含义 | 直觉 |
|------|------|------|
| $\min \frac{1}{2}\|w\|^2$ | 最大化间隔 | 路修宽点 |
| 间隔 = $\frac{2}{\|w\|}$ | 决策边界到支持向量的距离 | 路的宽度 |
| $K(x,z) = e^{-\gamma\|x-z\|^2}$ | RBF 核函数 | 相似度衰减 |
| $f(x) = \sum \alpha_i y_i K(x_i,x) + b$ | 决策函数 | 支持向量的加权投票 |

## 🔗 关联概念
- **前置知识**：线性回归、逻辑回归、过拟合与正则化
- **后续应用**：核方法、高斯过程、深度学习中的核视角
- **同类对比**：逻辑回归（概率分类）、随机森林（集成方法）、KNN（基于距离）
- **与随机森林的对比**：SVM 追求几何最优，随机森林追求统计平均

## 💡 记忆技巧

SVM = **S**upport（支持向量撑起边界）**V**ector（只有少数向量重要）**M**argin（最大化间隔）。

城市规划师类比：在两栋楼之间修最宽的马路——决策边界就是路中线，支持向量就是路两边的建筑外墙。
