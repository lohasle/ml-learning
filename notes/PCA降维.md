# PCA 降维

## 概述
PCA（Principal Component Analysis，主成分分析）是最经典的降维方法，通过线性变换将高维数据投影到方差最大的方向上。

## 核心机制

| 步骤 | 操作 | 数学表达 |
|------|------|----------|
| 1. 中心化 | 减均值 | X' = X - μ |
| 2. 协方差矩阵 | 特征间相关性 | C = X'ᵀX'/(n-1) |
| 3. 特征分解 | 求特征值和特征向量 | C·v = λ·v |
| 4. 投影 | 取前 k 个主成分 | Z = X'·Vₖ |

## 关键概念

- **主成分**：按方差从大到小排列的新坐标轴，彼此正交
- **explained_variance_ratio**：每个主成分保留的方差占比
- **累积方差**：前 k 个主成分总共保留了多少信息

## 关联概念
- [[KMeans]]：聚类 + 降维是无监督学习的两大分支
- [[SVM]]：PCA 降维后可以提高 SVM 的分类效果
- [[过拟合与正则化]]：降维本身是一种防止过拟合的手段
- 自编码器：深度学习中的非线性 PCA

## 实践要点
1. 降维前必须标准化（StandardScaler）
2. 先画累积方差曲线再选 k
3. 只能处理线性关系，非线性用 t-SNE/UMAP
4. 常用于：可视化（2D/3D）、压缩（减少特征）、去噪（丢弃小方差）
